Fetch news with their comments

SELECT
    n.slug,
    ..., -- some fields here
    array_agg(c) AS comments
  FROM
    news n
      LEFT JOIN comment c USING (news_slug)
  GROUP BY
      n.slug
  ;

Ranges

Are those two ranges contiguous ?

SELECT numrange(1.1, 2.2) -|- numrange(2.2, 3.3);

What's the next TV program after this one ?

Modifying the default projection

                      
public function getSelectFields($alias = null)
{
    $fields = parent::getSelectFields($alias);
    unset($fields['password']);
    $fields['age'] = sprintf(
        "age(%s)", 
        $this->aliasField('birthdate', $alias)
        );

    return $fields;
}
